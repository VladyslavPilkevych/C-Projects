<h1>StudentAnalyzer</h1>
<p>StudentAnalyzer is a C program for analyzing student data, including their grades, birth dates, and other information.</p>
<h2>Features</h2>
<ul>
<li><strong>Sum of grades:</strong> Displays information about students and their grades.</li>
<li><strong>Find student:</strong> Retrieves information about a student by name.</li>
<li><strong>Best test:</strong> Identifies the student with the highest grade.</li>
<li><strong>Gender:</strong> Finds the student with the highest grade for a specified gender.</li>
<li><strong>Region:</strong> Determines the best students by region.</li>
<li><strong>Birth year:</strong> Finds the student with the highest grade among those born after a specified year.</li>
<li><strong>Average grade:</strong> Displays the student with the highest average grade.</li>
<li><strong>Filter by grades:</strong> Outputs students with grades above a specified value.</li>
<li><strong>Add student:</strong> Allows adding a new student.</li>
<li><strong>Delete student:</strong> Allows deleting a student by name.</li>
</ul>
<h2>Usage</h2>
<pre>
./student_analyzer
</pre>
<p>You can enter commands to execute various functions of the program.</p>
<h2>Notes</h2>
<p>The program requires a <code>studenti.csv</code> file to store student data.</p>
<p>Make sure the data format in the file meets the program's requirements.</p>
